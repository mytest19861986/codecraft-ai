# چک‌لیست دمو MVP

## متغیرهای لازم

- `ADMIN_PASSWORD`
- `ADMIN_SESSION_SECRET`
- `DATABASE_URL`

## دستورهای قبل از دمو

از مسیر `apps/web` اجرا شود:

```bash
npm run lint
npm run typecheck
npm run build
npx prisma validate
```

## مسیر دمو در مرورگر

1. صفحه `/bootcamp` را باز کنید.
2. یک لید معتبر ثبت کنید.
3. همان شماره موبایل را دوباره ثبت کنید و پیام تکراری بودن را ببینید.
4. یک شماره موبایل نامعتبر وارد کنید و پیام اعتبارسنجی را ببینید.
5. صفحه `/admin/login` را باز کنید.
6. با رمز ادمین وارد شوید.
7. ثبت شدن لید را در `/admin/leads` بررسی کنید.
8. خروج را بزنید.

## ساخت لید تست

در این نسخه اسکریپت جداگانه‌ای برای seed وجود ندارد. ساده‌ترین روش ساخت لید تست همان ثبت فرم `/bootcamp` است تا مسیر واقعی دمو هم بررسی شود.

## خارج از محدوده دمو

SMS واقعی، اتصال تلگرام، پرداخت، OpenAI API و ویدیوی واقعی درس‌ها جزو نسخه‌های بعدی هستند و در دمو فعلی وجود ندارند.
